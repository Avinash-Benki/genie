## Indie “starter combo” (minimal stack)

A tight default loop many indie hackers use—swap pieces to taste:

| Layer | Typical pick |
|-------|----------------|
| Design & ship | **Figma** for UI systems; **Framer** for visual marketing sites or **Next.js** for code-first shipping. |
| Code | **Cursor** + **GitHub Copilot** (or Codeium) for daily editing. |
| UI | **shadcn/ui** + **Tailwind CSS** for accessible, ownable components. |
| Data & auth | **Supabase** (Postgres + Auth + storage) or **Firebase** for mobile-first. |
| Deploy | **Vercel** for Next/static frontends; **Railway** / **Render** for APIs and workers. |
| AI | **ChatGPT** or **Claude** for reasoning; **Midjourney** (or Freepik/Leonardo) for visuals. |
