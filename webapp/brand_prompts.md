# 🧠 1. Advanced Logo Concept Generator Prompt

```text
Act as a senior logo designer and creative director with 20+ years of experience creating iconic brand identities for global companies.

Your task is to develop distinct, strategic logo concepts that are meaningful, memorable, and scalable across digital and physical media.

Brand Information:
- Brand name: [YOUR BRAND NAME]
- Industry: [YOUR INDUSTRY]
- Brand personality: [DESCRIBE TRAITS]
- Target audience: [DESCRIBE DEMOGRAPHICS + PSYCHOLOGY]

Visual References:
- Brands whose visual identity I admire: [LIST 3]
- Brands whose visual identity I dislike: [LIST 3]
- Preferred design style: [minimal / bold / geometric / organic / retro / futuristic / etc.]

Task:
Generate 5 completely different logo concepts for this brand.

For each concept provide:
- Core visual idea and symbolic meaning
- Shape language (geometric, organic, abstract, etc.) and why it suits the brand
- Typography direction (serif, sans-serif, custom lettering, script, etc.)
- Immediate emotional impression the logo creates
- Why it resonates with the target audience
- How it scales across real-world use cases (business card, mobile app icon, large billboard)
- Why the concept will remain timeless rather than trend-driven

Final Step:
Evaluate all five concepts and explain which one you would choose if this were your own brand, including detailed reasoning.
```

# 🎨 2. Strategic Brand Color Palette Prompt

```text
Act as a brand strategist and color psychology expert specializing in how color influences perception, trust, and purchase behavior.

Brand Information:
- Brand name: [YOUR BRAND NAME]
- Industry: [YOUR INDUSTRY]
- Target audience: [Describe demographics + lifestyle + purchasing behavior]
- Primary emotion the brand should trigger: [trust / luxury / excitement / calm / innovation / etc.]

Market Context:
- Top competitors and their brand colors: [LIST]
- Colors I love: [LIST]
- Colors I dislike: [LIST]

Task:
Create a complete and strategically differentiated brand color palette.

Provide:

1. Primary Color
- Exact HEX code
- Psychological reasoning for why it fits the brand

2. Secondary Colors (2)
- HEX codes
- How they support and balance the primary color

3. Accent Color
- HEX code
- How it should be used for CTAs, highlights, and focus elements

4. Neutral Color
- HEX code
- Recommended use for backgrounds, UI elements, and typography

Strategic Explanation:
- How each color influences audience perception and emotion
- How the palette differentiates the brand from competitors

Practical Examples:
- How to apply the palette across website UI, social media, and product packaging
- Color combinations to avoid and the reasoning behind them
```

# 🔤 3. Brand Typography System Prompt

```text
Act as a typography specialist and brand designer with deep expertise in how fonts communicate identity, credibility, and emotion across digital and physical touchpoints.

Brand Information:
- Brand name: [YOUR BRAND NAME]
- Brand personality (5 words): [e.g., bold, modern, innovative, premium, friendly]
- Industry: [YOUR INDUSTRY]
- Target audience: [Describe demographics + behavior]
- Desired emotional tone of typography: [authority / friendliness / elegance / innovation / energy / etc.]
- Typography styles I admire: [LIST 3 BRANDS]

Task:
Design a complete typography system for the brand.

Provide:

1. Primary Display Font
- Font name
- Where it can be obtained (Google Fonts, Adobe Fonts, etc.)
- Why it matches the brand personality
- Recommended use (logo, hero headlines, large titles)

2. Secondary Body Font
- Font name
- Why it pairs well with the display font
- Recommended use for long-form content

3. Accent Font (optional)
- Font for quotes, highlights, or creative moments

Typography System Guidelines:
- Size hierarchy for H1, H2, H3, body text, captions
- Recommended letter spacing and line height
- Best practices for combining fonts without visual conflict
- Free alternative fonts for budget-friendly use
- The most common typography mistake brands in this industry make and how to avoid it
```

# 🧾 4. Advanced Typography System Prompt (More Detailed Version)

```text
Act as a typography expert and brand designer with deep experience building font systems for brands across digital and physical platforms.

Your goal is to design a cohesive typography system that communicates personality, builds credibility, and remains consistent across all brand touchpoints.

Brand Information:
- Brand name: [YOUR BRAND NAME]
- Brand personality (5 words): [DESCRIBE]
- Industry: [YOUR INDUSTRY]
- Target audience: [DESCRIBE THEM]
- Desired emotional tone: [authority, friendliness, innovation, elegance, energy, etc.]
- Brands whose typography I admire: [LIST 3]

Task:
Create a complete typography system for this brand.

Include:

Primary Display Font:
- Exact font name
- Where to access it (Google Fonts, Adobe Fonts, etc.)
- Why it reflects the brand personality
- Recommended use (logo, headlines, hero sections)

Secondary Body Font:
- Font name
- Why it pairs well with the display font
- Best use for long-form content and UI text

Accent Font:
- Font recommendation for quotes, highlights, or callouts
- Situations where it should be used sparingly

Typography Hierarchy:
Define a clear structure for:
- H1 headlines
- H2/H3 subheadings
- Body text
- Captions or microcopy

Include recommended font sizes, letter spacing, and line height for each level.

Usage Guidelines:
- Explain how to combine fonts so they remain visually balanced and consistent across layouts

Budget Alternatives:
- Provide free alternatives for each recommended font

Industry Insight:
- Identify one common typography mistake brands in this industry make and explain how this system avoids it
```
